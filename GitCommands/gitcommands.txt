Git is a version control helps to keep track of ones project versions and act as synchronize. 

*creating a new repository in local machine*

              		  git clone <link of git repository>


*To add file into GitHub *
			  git add . (adds all files present in that directory into gitub)
			  git add <filename.txt> (adds only filename.txt file)

*Git commit*
			  git commit -m "message"

*Git push*
			  git push 

*Git status*
 			  git status

*Git log*

*Git reset*
